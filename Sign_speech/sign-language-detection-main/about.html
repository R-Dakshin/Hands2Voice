Overview

This British Sign Language (BSL) Translator enables real-time translation of hand gestures into English letters using computer vision and machine learning.

How It Works

It captures video input through your webcam, processes each frame with a trained deep learning model, and identifies hand gestures that correspond to BSL characters. Predictions are displayed along with a confidence score.

Technologies Used

  • Python + Flask + Flask-SocketIO
  • TensorFlow + MobileNetV2
  • OpenCV for image processing
  • JavaScript for camera and socket integration
